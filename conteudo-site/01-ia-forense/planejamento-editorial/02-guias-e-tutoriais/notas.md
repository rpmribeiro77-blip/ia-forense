# Guias e tutoriais

## Objetivo da area

Criar conteudos passo a passo para advogados usarem IA em tarefas praticas, com entrada, comando, refinamento, resultado esperado e conferencia.

## Ideias de assuntos

- Como escrever um prompt juridico.
- Como pedir respostas em etapas.
- Como anexar documentos e pedir analise.
- Como revisar uma peticao.
- Como transformar decisao em roteiro de recurso.
- Como preparar audiencia.
- Como criar biblioteca pessoal de prompts.
- Como usar projetos, arquivos e canvas no ChatGPT.
- Como pedir que a IA faca perguntas antes de responder.
- Como montar um prompt de contestacao a partir da inicial.
- Como pedir contra-argumentos.
- Como pedir correcao textual com preservacao do sentido juridico.
- Como pedir fundamentacao legal sem confiar cegamente na resposta.
- Como transformar documentos em resumo para cliente.
- Como usar NotebookLM como complemento para documentos extensos.
- Como criar tutorial com entrada, comando, resultado e conferencia.
- Como usar Perplexity para pesquisa inicial com fontes verificaveis.
- Como validar pesquisa feita por Perplexity em fonte oficial.
- Como usar NotebookLM para montar linha do tempo de um processo.
- Como gerar perguntas frequentes ou guia de estudo a partir de documentos do caso.
- Como usar IA para Visual Law: fluxograma, quadro cronologico, mapa de argumentos.
- Como escolher entre ChatGPT, Claude, Gemini, NotebookLM e Perplexity para cada tarefa.
- Como preparar tutorial de prova digital: extrair fatos, organizar evidencias e listar cautelas.
- Tutorial: criar um Projeto no ChatGPT para um tema juridico continuado.
- Tutorial: usar Canvas para revisar artigo, parecer ou roteiro de video.
- Tutorial: usar Gemini Deep Research com cautelas e checagem.
- Tutorial: usar NotebookLM com documentos, videos do YouTube e fontes citadas.
- Tutorial: usar Codex para transformar notas em pagina do site.
