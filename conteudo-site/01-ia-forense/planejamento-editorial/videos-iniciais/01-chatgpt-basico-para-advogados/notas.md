# Video inicial - ChatGPT basico para advogados

## Objetivo do video

Apresentar o ChatGPT para advogados iniciantes, com foco em acesso, recursos basicos, usos juridicos iniciais e cautelas.

## Pontos para roteiro

- Como acessar o ChatGPT.
- O que e uma conversa com IA.
- Diferenca entre ChatGPT e Google.
- Como formular o primeiro pedido.
- Como pedir resposta em etapas.
- Como usar arquivos, quando disponivel.
- Como usar busca com citacoes.
- Como usar Projetos.
- Como usar Canvas.
- Como usar chat temporario.
- Controles de dados.
- Cuidados com sigilo, dados sensiveis e documentos de cliente.
- Diferenca entre chat comum, Projeto, GPT personalizado e chat temporario.
- Como nomear conversas para organizar a rotina.
- Como pedir que a IA explique limites da propria resposta.
- Como criar um projeto "Escritorio" ou "Processo Civil" no ChatGPT.
- Como usar Canvas para revisar um roteiro ou artigo juridico.
- Como explicar a ferramenta a partir das tarefas do advogado, nao dos botoes.

## Demonstracoes possiveis

- Transformar relato ficticio de cliente em linha do tempo.
- Pedir checklist de documentos.
- Revisar um paragrafo juridico ficticio.
- Pedir perguntas antes da resposta final.
- Pedir resumo para cliente em linguagem simples.
- Pedir comparacao entre duas teses em tabela.
- Pedir checklist de verificacao antes de usar uma resposta.
- Criar um projeto e colocar instrucoes de linguagem juridica clara.
- Abrir um canvas para revisar um texto.
- Fazer busca com citacoes e checar a fonte.
