# Video inicial - Codex App basico para advogados

## Objetivo do video

Apresentar o Codex App para advogados como ferramenta de trabalho com arquivos, sites, documentos e projetos, sem tratar como ferramenta exclusiva de programadores.

## Pontos para roteiro

- O que e o Codex.
- Como abrir uma pasta ou projeto.
- Como pedir uma tarefa em linguagem natural.
- Como o Codex le arquivos do projeto.
- Como revisar alteracoes propostas.
- Como usar para site juridico.
- Como usar para organizar conteudo.
- Como usar para criar paginas, listas e rascunhos.
- Cuidados com arquivos, backup, versionamento e sigilo.
- Diferenca entre ChatGPT e Codex.
- Como pedir para o Codex ler um projeto antes de editar.
- Como pedir alteracoes pequenas e revisaveis.
- Como usar Codex para planejamento editorial.
- Como usar Codex para transformar notas em pagina do site.
- Codex App como central para gerenciar agentes e tarefas em paralelo.
- Como usar Codex para trabalhar no site IA Forense sem precisar editar tudo manualmente.
- Como explicar aprovacoes, revisao e cautela antes de aceitar mudancas.

## Demonstracoes possiveis

- Criar uma pagina simples no site.
- Organizar notas de pesquisa em uma pasta.
- Transformar um material bruto em estrutura de artigo.
- Pedir ao Codex para revisar links e arquivos.
- Criar pastas de planejamento para conteudo juridico.
- Gerar primeira versao de roteiro a partir de notas.
- Pedir resumo do estado atual do projeto.
- Pedir para criar uma pasta de pesquisa e notas.
- Pedir para transformar radar de pesquisa em rascunho de artigo.
- Pedir para localizar links quebrados ou referencias faltantes.
